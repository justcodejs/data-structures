Linked List - C++

  • Create List

    List list;

    • Print all items in the list

      list.get();

    • Add item in the list

      list.push(int age, const char *name, const char *surname);

    • Pop the top item inserted in the list

      list.pop(&list.head);

    • Remove item in the list by index

      list.remove_by_index(int index);

    • Search an or more items in the list

      list.search(int age, const char *name, const char *surname);

    • Add item in the list

      list.destroy();
